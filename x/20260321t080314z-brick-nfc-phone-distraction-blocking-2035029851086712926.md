## Topic
Brick NFC device for intentional phone distraction blocking

## Research Summary
The image shows the Brick device, and the tweet reports strong short-term quality-of-life gains. Brick combines physical NFC interaction with app blocking, using friction as a behavior-change mechanism.

## Key Findings
- Physical friction (tap-to-enable/disable) can outperform purely software limits for some users.
- Early users report better control over compulsive phone checking.
- Value depends on consistent habit adoption, not hardware alone.

## Practical Relevance (for Gonzalo)
If you want fewer context switches during deep work, this is a practical intervention with low setup complexity.

## Risks / Caveats
Novelty effect may fade; users can bypass controls. Also depends on phone ecosystem compatibility.

## Recommended Next Steps
Run a 14-day experiment: define 2 blocked app sets (workday/evening), track subjective focus and task completion daily.
