## Topic

A long-running native macOS Markdown editor adding faster preview behavior, terminal shortcuts, better Mermaid support, and workflow polish for local note-taking.

## Research Summary

- MiaoYan is a real native Markdown application with an active GitHub project and release history.
- The release claim fits the project direction: preview rendering, diagram support, and local-first editing are central to the app’s appeal.
- Compared with browser-heavy note tools, the pitch here is speed, low friction, and an engineering-oriented workflow.

## Key Findings

- A mature native editor can still differentiate on responsiveness and keyboard-driven workflow, even in a crowded notes market.
- Preview and diagram improvements matter disproportionately for technical writing and knowledge-base use.
- The six-year polish claim rings true given the breadth of small quality-of-life features around local files.
