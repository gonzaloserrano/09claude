## Research Summary

## Key Findings
- Recurring jobs are strongest for deterministic maintenance tasks (docs sync, dependency checks, flaky-test triage).
- Guardrails (branch protections, test gates) are required before autonomous commit flows.
- Operational value comes from reducing context-switching and overnight progress.
