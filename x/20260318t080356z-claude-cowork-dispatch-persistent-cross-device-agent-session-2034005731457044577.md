## Topic
Claude Cowork Dispatch (persistent cross-device agent session)

## Research Summary
Dispatch extends Claude Cowork into a persistent, device-bridged thread where tasks can be queued from mobile and executed by the desktop session with local context and tools.

## Key Findings
- The original bookmark indicates strong practitioner demand around this workflow.
- Official documentation confirms the core mechanism and constraints.
- Independent sources suggest fast ecosystem adoption but uneven maturity in tooling and reliability.
