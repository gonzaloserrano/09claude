Claude Dispatch availability on Team plan (admin-enabled rollout)

## Research Summary
The bookmark is about Dispatch becoming available for Team plan workspaces but requiring an admin toggle. The practical implication is that feature readiness depends on workspace governance, not only per-user updates.

## Key Findings
- Dispatch access on Teams appears controlled at org level and may be off by default.
- Teams rollout reduces feature fragmentation between Max/Team users for mobile-to-desktop handoff workflows.
- Onboarding docs should include an “ask admin to enable Dispatch” step to avoid user confusion.
